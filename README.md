# Productivity Backend

## Description
The Productivity Backend is a RESTful API built with Spring Boot that facilitates task management for departments within an organization. It enables users to create, manage, and assign tasks to team members based on their department. The application leverages JWT for secure authentication and MongoDB for data storage.

## Features
- User authentication and authorization using JWT.
- Department management: create and manage departments.
- Task management: create, assign, and track tasks.
- Role-based access control to restrict certain operations.

## Technologies Used
- Spring Boot
- MongoDB
- JWT (JSON Web Tokens)
- Lombok
- Spring Security
